# Aether Pro - Weather Web Application
**PM Accelerator Technical Assessment #1 (Frontend)**
A modern, beautiful, and fully functional weather application built with React.js.
---
## ✨ Features
- **Real-time Weather** for any city or zip code
- **Current Location Detection** (GPS button)
- **5-Day Forecast** with detailed information
- **24-Hour Hourly Forecast**
- **Additional Details**: Sunrise, Sunset, Pressure, Visibility, Dew Point, UV Index, etc.
- **Dynamic Background** with beautiful mountain landscape that changes according to weather
- **Weather Effects**: Rain and Snow particle animations
- **Fully Responsive** design (Mobile + Desktop)
- **Loading & Error States** with good UX
- **Smooth Animations** and premium dark theme

---
## 🛠️ Tech Stack
- **Frontend**: React.js + Hooks
- **Styling**: CSS (with Glassmorphism + Custom Design Tokens)
- **API**: OpenWeatherMap API
- **Others**: Axios, Responsive Design, Geolocation API
---
## 📱 How to Use
1. Search any city name or zip code
2. Click the **📍 GPS** button to get weather of your current location
3. Enjoy beautiful weather insights with dynamic backgrounds and animations
